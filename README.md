# GameBBVA
Test to BBVA
Game created by Geo as a test for BBVA.

## Scripts
- npm ru build
- npm run build:watch
- npm run start

## Branches
 - Master : Branch for production
 - Dev : Branch for development


## npm install
Run the install all the packages necessaries to run the game
 - Transpiller : Babel
 - Bundler : Webpack
 - webpack-dev-server
 - html-webpack-plugin
 - eslint-webpack-plugin
 - html-loader
 - css-loader
 - sass-loader
 - style-loader
